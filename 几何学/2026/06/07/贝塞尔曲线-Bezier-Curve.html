概述

“有理”和“非有理”贝塞尔曲线在计算机图形学（如字体设计、CAD）和工业设计（如汽车建模）中非常核心的概念。

  • 非有理贝塞尔曲线：是最基本、最常用的形式。它由控制点和伯恩斯坦基函数定义，无法精确地表示圆锥曲线（如圆、椭圆、抛物线）。
  • 有理贝塞尔曲线：是非有理形式的推广。它为每个控制点引入了一个新的参数——权重。通过调整权重，它可以精确地表示圆锥曲线，并对曲线形状提供更高级的控制。

贝塞尔曲线是计算机图形学中描述平滑曲线的核心工具，它有许多重要且实用的性质。下面为你梳理了最关键的一些：

  • 端点性质：曲线的起点和终点分别与特征多边形的第一个和最后一个顶点严格重合。同时，曲线在起点处与特征多边形第一条边相切，在终点处与最后一条边相切。切线的方向由相邻两个控制点的连线方向决定。
  • 凸包性质：整条贝塞尔曲线会被完全包含在其所有控制点构成的凸包（能包含所有点的最小凸多边形）内部。这个性质对于曲线裁剪和碰撞检测很有帮助。
  • 几何不变性：曲线的形状只取决于控制点之间的相对位置，而与它们所在的坐标系（绝对位置）无关。对曲线做旋转、平移等仿射变换，等价于直接对所有控制点做相同的变换。
  • 变差缩减性：平面上任意一条直线与贝塞尔曲线的交点数量，不会多于该直线与特征多边形的交点数量。简单理解，就是曲线的“弯曲程度”不会比其控制多边形更剧烈，这让曲线显得更“光顺”。
  • 递推与分割：贝塞尔曲线可以用德卡斯特里奥（de Casteljau）算法递推定义。这个算法不仅稳定，还能将一条曲线在参数 t 处精确地分割成两段子曲线，而这两段子曲线仍然是贝塞尔曲线。
  • 对称性：保持所有控制点顺序颠倒（P0, P1,…, Pn 变为 Pn,…, P1, P0），得到的曲线与原曲线完全一致，只是参数 t 的方向相反。这意味着正向和反向绘制效果相同。
  • 全局支撑性：改变任意一个控制点的位置，都会影响整条曲线的形状（从 t=0 到 t=1 都会发生变化）。这既是优点（控制直观），也是缺点（局部调整困难）。
  • 仿射变换不变性：几何不变性的一个更精确的表述。即：要先对曲线做仿射变换，再计算其上的点；或者先计算曲线上的点，再对点做仿射变换，这两种方式得到的结果完全相同。
  • 直线保持性：如果所有控制点共线，那么生成的贝塞尔曲线就是一条直线段。

非有理贝塞尔曲线

这是大多数人最初学习的贝塞尔曲线形式。

定义

一条 n 阶非有理贝塞尔曲线由 n+1 个控制点 $P_0, P_1, …, P_n$ 定义，其公式为，其中，$B_{i,n}(t)$ 是伯恩斯坦基函数： \(C(t) = \sum_{i=0}^{n} B_{i,n}(t) P_i，B_{i,n}(t) = \binom{n}{i} t^i (1-t)^{n-i}\)

伯恩斯坦基函数

  • 每个都是一个关于 $t$ 的 $n$ 次多项式
  • 总共 $n + 1$ 个伯恩斯坦基函数构成了一组基底
  • 所有伯恩斯坦基函数的和为 1

性质

简单性： 定义和计算都非常直观。

仿射不变性： 对曲线进行旋转、平移、缩放等仿射变换，等价于直接对控制点进行相同的变换。

凸包性： 曲线完全位于其控制点构成的凸包内。

局限性：** 它无法精确地表示圆、椭圆、双曲线等圆锥曲线**。你只能用多段贝塞尔曲线来近似一个圆，但总会存在误差。

示例

线性贝塞尔曲线（1阶）：一条直线，$P_0$ 到 $P_1$。 二次贝塞尔曲线（2阶）：一个抛物线片段，由 $P_0$, $P_1$, $P_2$ 定义。 三次贝塞尔曲线（3阶）：最常用的类型，由 $P_0$, $P_1$, $P_2$, $P_3$ 定义，用于字体轮廓（TrueType字体）、UI设计等。

实现

#ifndef BEZIER_CURVE_HPP
#define BEZIER_CURVE_HPP

#include "../3rd/eigen/Eigen/Dense"

class BezierCurve {
private:
        std::vector<Eigen::Vector2d> _control_points;
        std::vector<Eigen::Vector2d> _derivative_points;
public:
        BezierCurve(const std::vector<Eigen::Vector2d>& control_points = {}) : _control_points(control_points) {
                if (control_points.size() <= 1) return;
                int n = control_points.size() - 1;
                _derivative_points.reserve(n);
                for (int i = 0; i < n; ++i) {
                        _derivative_points.push_back(n * (control_points[i + 1] - control_points[i]));
                }
        }

        // 使用德卡斯特里奥算法计算贝塞尔曲线点
        Eigen::Vector2d calculate_point(double t) const {
                return calculate(_control_points, t);
        }

        // 计算一阶导数（切向量）
        Eigen::Vector2d calculate_tangent(double t) const {
                 auto tangent = calculate(_derivative_points, t);
                 // 计算归一化切向量（单位切向量）
                 double norm = tangent.norm();
                 constexpr double epsilon = 1e-10;
                 return (tangent.norm() > epsilon) ? tangent.normalized() : Eigen::Vector2d::Zero();
        }

        // 均匀采样 UniformSampling
        std::vector<std::pair<Eigen::Vector2d, Eigen::Vector2d>> uniform_sampling(int num_samples = 100) const {
                std::vector<std::pair<Eigen::Vector2d, Eigen::Vector2d>> result;
                result.reserve(num_samples);

                for (int i = 0; i < num_samples; ++i) {
                        double t = static_cast<double>(i) / (num_samples - 1);
                        Eigen::Vector2d point = calculate_point(t);
                        Eigen::Vector2d tangent = calculate_tangent(t);
                        result.emplace_back(point, tangent);
                }

                return result;
        }

        // 自适应采样（基于切向量角度变化）AdaptiveSampling
        // angle_tolerance 角度容差（度）
        std::vector<std::pair<Eigen::Vector2d, Eigen::Vector2d>> adaptive_sampling(double angle_tolerance = 5.0, int max_depth = 10) const {
                std::vector<std::pair<Eigen::Vector2d, Eigen::Vector2d>> result;
                result.emplace_back(calculate_point(0.0), calculate_tangent(0.0));

                adaptive_sampling_recursive(result, 0.0, 1.0, angle_tolerance * M_PI / 180.0, max_depth, 0);

                result.emplace_back(calculate_point(1.0), calculate_tangent(1.0));

                return result;
        }
private:
        // 使用德卡斯特里奥算法计算贝塞尔曲线点
        Eigen::Vector2d calculate(const std::vector<Eigen::Vector2d>& control_points, double t) const {
                std::vector<Eigen::Vector2d> temp = control_points;
                for (int r = 1; r < temp.size(); ++r) {
                        for (int i = 0; i < temp.size() - r; ++i) {
                                temp[i] = (1 - t) * temp[i] + t * temp[i + 1];
                        }
                }
                return temp[0];
        }

        void adaptive_sampling_recursive(std::vector<std::pair<Eigen::Vector2d, Eigen::Vector2d>>& result, double t0, double t1,
                double angle_tolerance_rad, int max_depth, int depth) const {

                if (depth >= max_depth) return;

                double mid_t = (t0 + t1) / 2.0;
                Eigen::Vector2d p0 = calculate_point(t0);
                Eigen::Vector2d p1 = calculate_point(t1);
                Eigen::Vector2d pm = calculate_point(mid_t);

                Eigen::Vector2d t0_vec = calculate_tangent(t0);
                Eigen::Vector2d t1_vec = calculate_tangent(t1);
                Eigen::Vector2d tm_vec = calculate_tangent(mid_t);

                // 计算切向量之间的角度变化
                double angle_change1 = acos(t0_vec.dot(tm_vec));
                double angle_change2 = acos(tm_vec.dot(t1_vec));
                double max_angle_change = std::max(angle_change1, angle_change2);

                if (max_angle_change > angle_tolerance_rad) {
                        adaptive_sampling_recursive(result, t0, mid_t, angle_tolerance_rad, max_depth, depth + 1);
                        adaptive_sampling_recursive(result, mid_t, t1, angle_tolerance_rad, max_depth, depth + 1);
                } else {
                        result.emplace_back(pm, tm_vec);
                }
        }
};

#endif //BEZIER_CURVE_HPP

多项式曲线

#ifndef POLYNOMIAL_CURVE_HPP
#define POLYNOMIAL_CURVE_HPP

#include <vector>
#include "../3rd/eigen/Eigen/Dense"

class PolynomialCurve {
private:
        Eigen::VectorXd _coefficients;
public:
        // 直接通过系数描述多项式曲线
        PolynomialCurve(const std::vector<float>& coefficients) : _coefficients(coefficients.size()) {
                for (int i = 0; i < coefficients.size(); ++i) {
                        _coefficients(i) = coefficients[i];
                }
        }

        // 通过给定控制点及度数来拟合多项式曲线
        PolynomialCurve(const std::vector<Eigen::Vector2d>& control_points, int degree) :
                _coefficients(Eigen::VectorXd::Zero(degree + 1)) {
                int n = control_points.size();
                if (n < _coefficients.size()) return;

                // 构建Vandermonde矩阵X
                Eigen::MatrixXd X(n, _coefficients.size());
                Eigen::VectorXd Y(n);

                for (int i = 0; i < n; ++i) {
                        float x = control_points[i].x();
                        for (int j = 0; j < _coefficients.size(); ++j) {
                                X(i, j) = std::pow(x, _coefficients.size() - j - 1);  // 从高次到低次排列
                        }
                        Y(i) = control_points[i].y();
                }

                // 使用最小二乘法求解系数
                _coefficients = X.bdcSvd(Eigen::ComputeThinU | Eigen::ComputeThinV).solve(Y);
        }

        float calculate_y(float x) const {
                float y = 0.0;
                for (int i = 0; i < _coefficients.size(); ++i) {
                        y = y * x + _coefficients[i];
                }
                return y;
        }

        float calculate_slope(float x) const {
                float slope = 0.0;
                int degree = _coefficients.size() - 1;
                for (int i = 0; i < degree; ++i) {
                        slope = slope * x + _coefficients[i] * (degree - i);
                }
                return slope;
        }

        std::vector<std::vector<float>> sampling(double x_start, double x_end, int num_samples) {
                std::vector<std::vector<float>> samples;
                samples.reserve(num_samples);

                double step = (x_end - x_start) / (num_samples - 1);
                for (int i = 0; i < num_samples; ++i) {
                        float x = x_start + i * step;
                        float slope = calculate_slope(x);
                        float dist = std::sqrt(1.0 * 1.0 + slope * slope);
                        samples.push_back({ x, calculate_y(x), 1 / dist, slope / dist });
                }

                return samples;
        }
};

#endif // POLYNOMIAL_CURVE_HPP

有理贝塞尔曲线

为了解决非有理形式无法表示圆锥曲线的根本性限制，有理贝塞尔曲线被引入。

定义

一条 n 阶有理贝塞尔曲线同样由 n+1 个控制点 $P_0, P_1, …, P_n$ 定义，但每个控制点都有一个附加的权重 $w_i$（通常为标量正实数）。其公式为： \(C(t) = \frac{ \sum_{i=0}^{n} w_i B_{i,n}(t) P_i }{ \sum_{i=0}^{n} w_i B_{i,n}(t) }\)

你可以把这个公式理解为： 分子：一个加权的控制点求和（类似于非有理形式，但每个点乘了权重）。 分母：一个权重的求和，其作用是将曲线“归一化”，确保计算结果是正确的。

权重的作用： 权重 $w_i$ 控制了控制点 $P_i$ 对曲线形状的吸引力。 增大某个控制点的权重 $w_i$，会使曲线被“拉向”该控制点 $P_i$。 减小某个控制点的权重 $w_i$，会使曲线“远离”该控制点 $P_i$。 当所有权重 $w_i$ 都相等时，分母是一个常数，有理贝塞尔曲线就退化为非有理贝塞尔曲线。

核心优势：

  • 精确表示圆锥曲线：这是它最重要的用途。通过精心选择控制点和权重，可以用一段有理二次贝塞尔曲线（只有3个控制点）精确地表示一段圆弧、椭圆弧或抛物线。 * 例如，一个90度的圆弧可以用一段权重为 $1, \frac{\sqrt{2}}{2}, 1$ 的有理二次贝塞尔曲线来精确表示。
  • 更强的造型能力：权重提供了除控制点位置之外的另一个控制维度，允许设计师更精细地调整曲线形状，而无需增加控制点。

示例

字体轮廓：OpenType字体中的 CFF（PostScript）轮廓 就使用有理贝塞尔曲线（更具体地说，是NURBS的一个子集）来定义，这使得它们能更精确地描述曲线。 计算机辅助设计：几乎所有专业的CAD系统（如SolidWorks, CATIA, AutoCAD）都基于 NURBS（非均匀有理B样条），它是有理贝塞尔曲线更一般化的形式。汽车、飞机、轮船的复杂曲面都是用NURBS建模的。 工业设计：任何需要精确表示圆形、圆角、过渡面的地方都会用到它。

拓展 NURBS

NURBS（Non-Uniform Rational B-Spline，非均匀有理B样条） 是贝塞尔曲线理论的最终集大成者。你可以这样理解它们的关系：

贝塞尔曲线是B样条的基础。 有理贝塞尔曲线是NURBS的基础。 NURBS = 非均匀（节点向量） + 有理（带权重） + B样条基函数。

NURBS继承了有理贝塞尔曲线所有优点的同时，还解决了贝塞尔曲线局部修改性差（移动一个控制点会影响整条曲线）的问题。因此，它成为了工业建模领域事实上的标准。

结论

如果你只是在设计一个UI图标或一个简单的矢量图形，非有理贝塞尔曲线（尤其是三次的）完全够用，且更简单高效。

如果你需要精确地创建一个圆、一个汽车外壳或一个复杂的有机曲面，有理贝塞尔曲线（或其更通用的形式NURBS）是必不可少的工具，因为它提供了所需的数学精度和造型灵活性。
