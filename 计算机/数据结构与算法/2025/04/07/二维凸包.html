想象一块钉着许多钉子的木板，如果拿一根橡皮筋，让它把所有钉子都围在内部，然后松手，橡皮筋收缩后会紧紧绷在最外围的某些钉子上。这根橡皮筋最终形成的形状，就是这些钉子（点集）的凸包。

定义

凸集：一个集合中，任意两点连成的线段完全落在此集合内。

凸多边形：所有内角大小都在 $[0, \pi]$ 范围内的简单多边形。

凸包：平面上能包含所有给定点的最小凸多边形叫做凸包。

  • 在包含所有点的多边形中（可能是凸多边形，也可能不是凸多边形），凸包的周长是最短的
  • 小于 4 个点直接就是凸包，点和线短可以看作是特殊的凸包

算法

Gift Wrapping算法 (Jarvis步进法)

这是最符合直觉的算法，就像用双手包裹礼物。

算法步骤：

  • 从一个必然在凸包上的点（例如最左边的点）开始。
  • 然后，寻找下一个点，使得所有其他点都在当前点与这个点的右侧（叉积判断），这个点就是新的凸包顶点。
  • 重复这个过程，直到回到起点。

复杂度：$O(n·h)$，其中 $n$ 是总点数，$h$ 是凸包顶点数

Graham Scan算法

这是最经典的算法，巧妙利用了排序和栈结构

算法步骤：

  • 找基点：找到y坐标最小的点（如果相同取最左），记为 $p_0$
  • 排序：将其余所有点按相对于P0的极角从小到大排序。 扫描：维护一个栈。将 $p_0$ 和排序后的前两个点压栈。然后依次处理每个点，检查栈顶两点与当前点构成的转向。
    • 如果是左转（叉积>0），则将当前点压栈；
    • 如果是右转（叉积<0），则说明栈顶点是“凹”的，需要弹出，继续检查，直到可以左转为止。

时间复杂度：$O(nlog(n))$，主要耗时在排序上。这是最优的复杂度下限。

空间复杂度：空间复杂度：$O(n)$

Andrew‘s Monotone Chain算法

这是Graham Scan的改进版，更简洁、数值稳定性更好。

核心原理

将凸包分为 “上凸壳” 和 “下凸壳” 分别构造，最后合并起来。

算法步骤：

  • 排序：将所有点按 x 坐标升序（x 相同按 y 升序）排列，第一个点和最后一个点一定在凸包上
  • 构造下凸壳：从左到右扫描，同样利用栈和左转/右转判定（对于下凸壳，我们维护一种转向一致性）
  • 构造上凸壳：从右到左扫描
  • 合并：将上下凸壳合并（注意去除重复的端点）

实现

当前点与栈顶两个点的外积小于 0 时，需要不断弹出栈顶，直到剩下一个元素为止

// (o, a) x (o, b)
template<typename T>
T cross(const std::pair<T, T>& o, const std::pair<T, T>& a, const std::pair<T, T>& b) {
    return (a.first - o.first) * (b.second - o.second) -
           (b.first - o.first) * (a.second - o.second);
}

template<typename T>
std::vector<std::pair<T, T>> andrew(std::vector<std::pair<T, T>> pts) {
    // 点数 ≤2 时手动构造闭合形式
    if (pts.empty()) return {};
    if (pts.size() == 1) return {pts[0], pts[0]};               // [p, p]
    if (pts.size() == 2) return {pts[0], pts[1], pts[0]};       // [p0, p1, p0]

    // 按 x 升序，x 相同时按 y 升序
    std::sort(pts.begin(), pts.end());

    std::vector<std::pair<T, T>> hull;
    
    auto push_convex = [&hull](const size_t lower_size, const std::pair<T, T>& pt) {
        while (hull.size() > lower_size && cross(hull[hull.size() - 2], hull.back(), pt) <= 0) hull.pop_back();
        hull.push_back(pt);
    };

    // 下凸壳 (从左到右)
    for (size_t i = 0; i < pts.size(); ++i) push_convex(1, pts[i]);

    // 上凸壳 (从右到左)，保留下凸壳部分不被弹出
    size_t lower_size = hull.size();
    for (size_t i = pts.size() - 1; i-- > 0;) push_convex(lower_size, pts[i]);

    // 此时 hull 的最后一个点与第一个点相同，直接返回（闭合凸包）
    return hull;
}

复杂度

时间复杂度：$O(nlog(n))$

空间复杂度：$O(O(n))$
