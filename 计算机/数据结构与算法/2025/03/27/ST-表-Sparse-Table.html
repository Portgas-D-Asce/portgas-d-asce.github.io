概述

ST 表是用于解决 可重复贡献 类型区间类问题的数据结构。

  • 不支持动态修改
  • 支持任意区间查询，且时间复杂度为常数！！！
  • 空间复杂度也还不错

使用 ST 表的前提是：满足可重复贡献！！！常见的可重复贡献问题

  • 最大/小值
  • 按位与
  • 按位或
  • GCD/LCM

构造

ST 表通过动态规划方式构建。

状态定义

\[dp[i][j]：区间 [i, i + 2^j - 1] 的最大值\]

状态转移

对区间进行拆分：

\[[i, i + 2^j - 1] = [i, i + 2^{j - 1} - 1] + [i + 2^{j - 1}, i + 2^{j - 1} + 2^{j - 1} - 1]\]

其中：

  • $[i, i + 2^j - 1]$ 区间的最大值为 $dp[i][j]$
  • $[i, i + 2^{j - 1} - 1]$ 区间的最大值为 $dp[i][j - 1]$
  • $[i + 2^{j - 1}, i + 2^{j - 1} + 2^{j - 1} - 1]$ 区间的最大值为 $dp[i + 2^{j - 1}][j - 1]$

故，状态转移方程为：

\[dp[i][j] = max(dp[i][j - 1], dp[i + 2^{j - 1}][j - 1])\]

查询

设查询区间 $[p, r]$，区间长度 $r - p + 1$。

求 $s$ 使得：

\[((r - p + 1) / 2) < 2 ^ s <= r - p + 1\]

使用 $s$ 将查询区间划分为两个长度均为 $2^s$ 的区间（一定具有重叠区域）：

\[[p, r] = [p, p + 2^s - 1] \cup [r - 2^s + 1, r]\]

故，查询结果定义为：

\[max(dp[p][s], dp[r - 2^s + 1][s])\]

_op(_dp[p][s], _dp[r + 1 - (1 « s)][s]);

// std::maximum is function template, not types of functors
template<typename T> struct Max  {
    constexpr T operator()(const T& lhs, const T& rhs) const { return std::max(lhs, rhs); }
};

// std::min is function template, not types of functors
template<typename T> struct Min {
    constexpr T operator()(const T& lhs, const T& rhs) const { return std::min(lhs, rhs);}
};

// std::gcd(c++17) is function template, not types of functors
template<typename T> struct Gcd {
    constexpr T operator()(const T& lhs, const T& rhs) const { return std::gcd(lhs, rhs);}
};

// std::lcm(c++17) is function template, not types of functors
template<typename T> struct Lcm {
    constexpr T operator()(const T& lhs, const T& rhs) const { return std::lcm(lhs, rhs);}
};

template<template<typename> class Op, typename T>
class SparseTable {
public:
    explicit SparseTable(const std::vector<T>& nums) : _dp(), _op() {
        int n = nums.size();
        // passing zero to clz(), which is not a valid argument
        // how is it come: 2 ^ j - 1 >= n - 1
        int m = (n <= 1 ? 0 : 32 - __builtin_clz(n - 1)) + 1;
        _dp = std::vector<std::vector<T>>(n, std::vector<T>(m));

        for(int i = n - 1; i >= 0; --i) {
            _dp[i][0] = nums[i];
            for(int j = 1; j < m; ++j) {
                _dp[i][j] = _dp[i][j - 1];
                int k = i + (1 << (j - 1));
                if(k < n) _dp[i][j] = _op(_dp[i][j], _dp[k][j - 1]);
            }
        }
    }
    
    // 0-indexed [p, r]
    T query(int p, int r) const {
        int n = _dp.size();
        if(p > r || p < 0 || r >= n) throw std::out_of_range("index out of range or p > r");
        
        if(p == r) return _dp[p][0];
        
        // how is it come: r + 1 - 2 ^ s <= p + 2 ^ s - 1 + 1
        int s = 31 - __builtin_clz(r - p);
        return _op(_dp[p][s], _dp[r + 1 - (1 << s)][s]);
    }
private:
    std::vector<std::vector<T>> _dp;
    Op<T> _op;
};

示例

SparseTable<std::bit_and, int> st(nums);

SparseTable<std::bit_or, int> st(nums);

SparseTable<Max, int> st(nums);

SparseTable<Min, int> st(nums);

SparseTable<Gcd, int> st(nums);

SparseTable<Lcm, int> st(nums);

复杂度

时间复杂度：

  • 构建复杂度 $O(nlog(n))$
  • 查询复杂度 $O(1)$

空间复杂度：$O(nlog(n))$
